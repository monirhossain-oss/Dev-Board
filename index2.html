Question-1: What are the different ways to select an element in the DOM?

Answer: There are several ways to select an element in the DOM: getElementById() selects an element by its ID, getElementsByClassName() selects elements by class name, and getElementsByTagName() selects elements by tag name. querySelector() selects the first matching element using a CSS selector, while querySelectorAll() selects all matching elements. Among these, querySelector() and querySelectorAll() are the most flexible since they allow selection based on complex CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: `innerHTML`, `innerText`, and `textContent` are used to get or modify content inside an element, but they work differently. `innerHTML` retrieves or sets the HTML inside an element, meaning it processes and renders HTML tags. `innerText` gets or sets only the visible text, ignoring hidden elements and considering CSS styles. `textContent` retrieves or sets all text inside an element, including hidden content, without processing HTML. `textContent` is faster and safer than `innerHTML` since it does not execute scripts or render HTML.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique in JavaScript where a parent element listens for events on its child elements using event bubbling. Instead of adding event listeners to multiple child elements, a single event listener is placed on a common ancestor. When an event occurs on a child, it propagates up the DOM tree, and the parent handles it using `event.target` to determine which child triggered the event. This improves performance and is useful for dynamically added elements.
